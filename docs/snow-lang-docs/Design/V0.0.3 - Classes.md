# Details and Design
## Type Inspection
- The `__typedesc` object
	- `fn isClass(): bool`: Returns true if the type is a class
### Lifetime Safety 
- Rule 1:
	1) Classes cannot hold references as member variables.
		```
		let x = SomeClass();
		{
			let y = AnotherClass();
			x.mY = ref y; // Assume that x has a variable for AnotherClass type called mY
		} // y goes out of scope here, x.mY contans a reference to this variable that is now pointing at invalid memory
		// The lifetime of x is greater than y in this situation
		```
		- Due to this, classes cannot store references of any kind as member variables. That is a pretty big issue though, how do you hold other classes that are shared between multiple objects? Instead of a raw reference, it needs to be a smart reference such as a `sref` (shared reference)
